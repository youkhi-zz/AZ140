Contoso 회사

  functionality, scalability, security, cost를 고려해서 AVD를 사용하기 적합한지 확인해야 함
  
AVD는 Azure에서 구동되는 desktop, app 가상화 솔루션
  managed service로 인프라 가상화 지원
  VM관리 지원
  Identity 관리용 솔루션으로 AAD를 사용
  Microsoft Endpoint Manager같은 기존 툴 지원
